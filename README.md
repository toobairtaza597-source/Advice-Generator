# Random Advice Generator

A lightweight single-page web app that displays random advice fetched from the [Advice Slip API](https://api.adviceslip.com/).

## Files

- `index.html` – page structure  
- `style.css` – modern, responsive styling  
- `script.js` – fetches advice, updates DOM, handles loading states  

## How to Run

1. Clone or download the three files into the same folder.  
2. Open `index.html` in your browser (or use VS Code Live Server).  

Click the **Get Advice** button – a new random tip appears instantly.

## Tech Stack

HTML5, CSS3, JavaScript (ES6+), Fetch API
